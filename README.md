## 3filmy-v2
Simple movies explorer
[Live](http://3filmy-v2.vercel.app/ "3filmy-v2.vercel.app")

## Getting started
Clone
```bash
git clone https://github.com/ThisGuh/3filmy-v2 
```
Install
```bash
npm install
```
Start
```bash
npm run dev
```
